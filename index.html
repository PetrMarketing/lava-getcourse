Вебхук LavaTop

LavaTop API не отдаёт оплаты через чекаут. Вебхук получает все платежи в реальном времени.

Как настроить в LavaTop:
1. Интеграции → Public API → три точки (...) рядом с API ключом
2. "Добавить Webhook"
3. Вставить URL выше
4. Тип: Результат оплаты (для подписок — добавьте второй: "Регулярный платеж")
5. Аутентификация: любой метод (Basic Auth или API Key)
